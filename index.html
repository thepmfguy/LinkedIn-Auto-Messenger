Detailed Instructions

1. Prepare LinkedIn

Visit your LinkedIn job post and navigate to the view where all candidates are shown

2. Open Developer Console

Press F12 or right-click and select "Inspect" to open Developer Tools, then click on the "Console" tab

3. Enable Pasting

If this is your first time, type "allow pasting" in the console when prompted

4. Run the Script

Paste the generated script and press Enter to start the automation

5. Multiple Pages

Remember to run the script again when you navigate to the next page of applicants
